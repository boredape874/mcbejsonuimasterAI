# External Example Sources

These sources are not the primary authority for vanilla behavior, but they are useful implementation references.

## `LeGend077/json-ui-examples`

Upstream:

- <https://github.com/LeGend077/json-ui-examples>

Local mirror:

- `references/external/json-ui-examples/`

Best for:

- custom progress bars
- slider examples
- toggle examples
- scroll panel templates
- layout size and offset snippets

## `Refaltor77/EasyUIBuilder`

Upstream:

- <https://github.com/Refaltor77/EasyUIBuilder>

Local mirror:

- `references/external/EasyUIBuilder/`

Best for:

- element-by-element sample files
- builder-generated JSON UI structure
- component naming patterns
- validating what a JSON UI builder emits for panels, labels, buttons, grids, sliders, and toggles

## How to use these sources

- Use them for pattern borrowing, not vanilla truth
- Use them after local sample packs if the request is about implementation templates
- Use them before inventing a new pattern from scratch
