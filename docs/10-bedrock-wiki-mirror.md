## How to use the mirror

Use the local mirror when:

- the skill must remain portable inside this GitHub repository
- the task needs a specific Bedrock Wiki page that should be readable offline
- you want exact topic routing without leaving the repository

Use the live site when:

- you need the newest update and are explicitly browsing

Primary live page:

- <https://wiki.bedrock.dev/json-ui/json-ui-documentation>
