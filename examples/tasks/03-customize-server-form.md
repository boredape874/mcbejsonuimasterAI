# Task: Customize Server Form

## Goal

Turn a default Bedrock server form into a more structured custom layout while keeping routing maintainable.

## Recommended skills

- `mcbe-json-ui-server-forms`
- `mcbe-json-ui-patterns`

## Inspect first

- `server_form.json`
- any custom chest or form helper files
- title-prefix or form-ID routing logic

## Expected workflow

1. find existing server-form factory structure
2. decide whether to use title routing, form identifiers, or a factory split
3. keep the patch minimal and explicit
4. confirm all shared templates used by the new form

## Expected result

- exact routing approach
- exact file patch
- any required protocol conventions from PMMP or Script API
