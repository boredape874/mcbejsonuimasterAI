# Pack Analysis Prompts

## 1. Analyze a full pack

```text
Use mcbe-json-ui-master.
Analyze this Bedrock resource pack end to end.
Produce a file-level map of entry points, shared templates, HUD systems, form systems, texture dependencies, and any server-side text protocols implied by the UI.
```

## 2. Learn from a working sample

```text
Use mcbe-json-ui-samples and mcbe-json-ui-patterns.
Reverse engineer this working sample pack and extract the reusable patterns, the required data channels, and the minimum set of files needed to transplant the behavior into another pack.
```
