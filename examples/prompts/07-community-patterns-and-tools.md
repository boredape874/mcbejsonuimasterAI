# Community Patterns And Tool Prompts

## 1. String splitting vs slicing

```text
Use mcbe-json-ui-logic and mcbe-json-ui-patterns.
Compare separator-based JSON UI string splitting with fixed-width slicing for this payload.
Tell me which approach is safer, what server-side preprocessing is needed, and what limitations apply.
```

## 2. AUX ID workflow

```text
Use mcbe-json-ui-tooling.
Explain how bedrock-auxgen fits into a Bedrock JSON UI or PMMP item rendering workflow.
Show when to use AUX IDs and when to use verified texture paths instead.
```

## 3. Dumper workflow

```text
Use mcbe-json-ui-tooling and mcbe-json-ui-vanilla-assets.
Use JSON-UI-Dumper as a discovery aid, then explain what should be confirmed against Ztech or bedrock-samples before patching the pack.
```

## 4. StarLibV2 architecture reference

```text
Use mcbe-json-ui-tooling and mcbe-json-ui-server-forms.
Use StarLibV2 only as an architecture reference for dynamic form organization.
Do not copy GPL code unless licensing is intentionally accepted.
```
