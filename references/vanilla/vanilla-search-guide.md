# Vanilla Search Guide

Use these search targets against `ZtechNetwork/MCBVanillaResourcePack`.

## UI textures

Search paths:

- `textures/ui/`

Search terms:

- `arrow`
- `button`
- `close`
- `edit`
- `lock`
- `import`
- `hud`
- `tips`
- `background`

## Item icons

Search file:

- `textures/item_texture.json`

Look for:

- texture keys
- atlas names
- item short names

## Block icons

Search file:

- `textures/terrain_texture.json`

Look for:

- terrain texture keys
- block texture aliases

## Screen files

Search paths:

- `ui/`

Look for:

- screen file names
- binding names
- control names
- `_ui_defs.json`
- `_global_variables.json`
