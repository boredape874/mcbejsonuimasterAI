# Source Overview

This local skill was created from the public repository:

- `https://github.com/XxVoidicxX/mcbe-ui-codex`

What was clearly present at creation time:

- A repository README describing the project as a verified MCBE UI reference for developers and AI tools.
- `docs/MCBE_UI_Icon_Reference.md`
- icon preview/image folders under `icons/`

What this local skill currently relies on:

- The README summary for intent and scope.
- The icon reference markdown for verified vanilla texture paths.

Important limitation:

- The repository README mentions additional Script API and JSON UI references, but the cloned snapshot used for this skill only clearly contained the icon reference document. This skill therefore focuses on the verified texture/icon lookup use case and does not claim to bundle missing documents.
