# Local Examples

Curated, compact examples copied from a local resource-pack archive.

These mirrors exist so Codex can inspect a focused working pattern without loading the full local archive.

## Included mirrors

- `rpg-hud`
  - title-driven RPG HUD
  - preserved title payloads
  - multiple animated bars
- `npc-dialogue`
  - `server_form.json` dialogue layout
  - BP script bridge for form data
- `multi-animated-progress`
  - compact multiple animated progress bar example

## Usage rule

Load only the mirror matching the task.

For broader local archive search, read:

- `../../docs/28-local-example-mining.md`

Some local archive folders contain third-party packs or files with explicit reuse restrictions. Those are documented as concepts only and should not be copied into public outputs without permission.
